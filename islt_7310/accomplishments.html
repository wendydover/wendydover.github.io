Library Guides

Using the SpringShare suite of tools, I have created library guides to assist students and faculty to access, evaluate, and utilize library print and online resources as well as other materials found on the open web.

Nursing Tutorial

CINAHL is a discipline specific database and not always intuitive. I created Gulf Coast State College Library tutorials to help nursing students use all the features of this database.

Designed For-Credit Course

Research Skills for College Students is a for-credit course based on the ACRL Framework and provides a step-by-step guide for selecting a topic, formulating a thesis, generating keywords, researching the topic, evaluating resources, and using resources ethically.
